Firebase підключено
Firebase Realtime Database — безкоштовно, синхронізує дані між телефоном і ноутбуком у реальному часі.

  1. Відкрий console.firebase.google.com
  2. Створи проект (або обери існуючий)
  3. Меню зліва: Build → Realtime Database → Create
  4. Обери регіон і режим Start in test mode
  5. Меню зліва: Project Overview → Web (</>) → Register app
  6. Скопіюй блок firebaseConfig — лише фігурні дужки з даними
  7. Встав нижче і натисни «Підключити»
